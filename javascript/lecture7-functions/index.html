JavaScript Functions

Note: Open the browser console (F12 or right-click > Inspect > Console) to see the output of the JavaScript code.

What are Functions?

Functions are reusable blocks of code designed to perform specific tasks. They help us:

  • Organize code into logical blocks
  • Reduce code duplication
  • Make code more maintainable and readable
  • Create reusable components

Function Syntax

Basic syntax of a JavaScript function:

function functionName(parameter1, parameter2) {
  // Code to be executed
  return result; // Optional return statement
}
