JavaScript Introduction

Note: Open the browser console (F12 or right-click > Inspect > Console) to see the output of the JavaScript code.

What is JavaScript?

JavaScript is a programming language that enables interactive web pages. It is an essential part of web applications, and nearly all modern websites use it.

Key Features of JavaScript:

  • Client-side scripting language
  • Interpreted (not compiled)
  • Object-oriented
  • Event-driven
  • Lightweight and versatile

Console.log()

The console.log() function is used to output messages to the browser's console. It's one of the most basic debugging tools in JavaScript. 

// Basic usage
console.log("Hello, World!");

// Printing variables
let name = "John";
console.log(name);

// Printing multiple values
console.log("Name:", name, "Age:", 25);

Variables

Variables are containers for storing data values. In JavaScript, we use let, const, or var to declare variables. 

// Declaring variables
let name = "John";      // String
let age = 25;           // Number
let isStudent = true;   // Boolean

// Constants (cannot be reassigned)
const PI = 3.14159;
const MAX_USERS = 100;

Data Types

JavaScript has several built-in data types:

Primitive Data Types:

  • String: "Hello", 'World'
  • Number: 42, 3.14
  • Boolean: true, false
  • Undefined: undefined
  • Null: null
  • Symbol: Symbol("id")
  • BigInt: BigInt(9007199254740991)

Complex Data Type:

  • Object: { name: "John", age: 25 } 
// Checking the type of a value
console.log(typeof "Hello");     // "string"
console.log(typeof 42);          // "number"
console.log(typeof true);        // "boolean"
console.log(typeof undefined);   // "undefined"
console.log(typeof null);        // "object" (this is a known JavaScript quirk)
console.log(typeof {});          // "object"

Getting Started with JavaScript

To include JavaScript in an HTML page, you can:

  1. Use the <script> tag directly in your HTML
  2. Link to an external JavaScript file (recommended)

Internal JavaScript:

<script>
  console.log("Hello from internal JavaScript!");
</script>

External JavaScript (what we're using):

<script src="index.js"></script>
