JavaScript Objects and String Methods

Note: Open the browser console (F12 or right-click > Inspect > Console) to see the output of the JavaScript code.

JavaScript Objects

Objects are collections of related data and functionality stored as key-value pairs.

Object Syntax

let person = {
  name: "John",
  age: 30,
  isStudent: false,
  address: {
    city: "Boston",
    country: "USA"
  },
  greet: function() {
    return "Hello!";
  }
};

Accessing Object Properties

There are two ways to access object properties:

  • Dot Notation: person.name
  • Bracket Notation: person["name"]

String Methods

JavaScript provides many built-in methods to manipulate strings.

Method Description Example
length Returns the length of a string "Hello".length → 5
toUpperCase() Converts to uppercase "hello".toUpperCase() → "HELLO"
toLowerCase() Converts to lowercase "HELLO".toLowerCase() → "hello"
includes() Checks if string contains a value "Hello".includes("lo") → true
slice() Extracts a part of a string "Hello".slice(1, 3) → "el"
