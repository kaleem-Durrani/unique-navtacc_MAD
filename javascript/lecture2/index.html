Arithmetic Operators

JavaScript provides several operators for performing mathematical calculations.

Operator Name Example Result
+ Addition 5 + 3 8
- Subtraction 5 - 3 2
* Multiplication 5 * 3 15
/ Division 6 / 3 2
** Exponentiation 2 ** 3 8
% Modulus (Remainder) 7 % 3 1

Data Types

JavaScript has several built-in data types:

  • String - Text values (e.g., "Hello")
  • Number - Numeric values (e.g., 42, 3.14)
  • Boolean - true or false
  • Undefined - A variable that has been declared but not assigned a value
  • Null - Represents the intentional absence of any value
  • Object - Collections of related data
  • Symbol - Unique and immutable primitive values

You can check the type of a value using the typeof operator: 

typeof "Hello"   // "string"
typeof 42        // "number"
typeof true      // "boolean"

Variables

Variables are containers for storing data values. In JavaScript, we declare variables using:

  • let - For variables that can be reassigned
  • const - For variables that cannot be reassigned
  • var - Older way to declare variables (not recommended for modern code)

Declaring Variables

// Using let
let name = "John";
let age = 25;
let isStudent = true;

// Using const
const PI = 3.14159;
const MAX_SIZE = 100;

Reassigning Variables

let score = 10;
console.log(score);  // 10

score = 20;
console.log(score);  // 20

// JavaScript is dynamically typed
let value = "hello";
value = 42;  // This is allowed!

String Concatenation and Template Literals

There are multiple ways to combine strings in JavaScript:

Traditional Concatenation

let name = "John";
let age = 25;

// Using the + operator
let message = "My name is " + name + " and I am " + age + " years old.";
console.log(message);

Template Literals (ES6)

// Using backticks (`) and ${expression} syntax
let message = `My name is ${name} and I am ${age} years old.`;
console.log(message);

Variable Swapping

Sometimes we need to swap the values of two variables:

Method 1: Using a Temporary Variable

let a = 5;
let b = 10;

// Swap using a temporary variable
let temp = a;
a = b;
b = temp;

console.log(a, b);  // 10, 5

Method 2: Using Destructuring Assignment (ES6)

let x = 5;
let y = 10;

// Swap using destructuring
[x, y] = [y, x];

console.log(x, y);  // 10, 5
