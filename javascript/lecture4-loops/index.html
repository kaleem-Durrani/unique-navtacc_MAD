JavaScript Loops

What are Loops?

Loops are used to execute a block of code repeatedly until a specified condition is met. They are essential for automating repetitive tasks and processing collections of data.

Loop Type Description Best Used When
for Runs code a specific number of times You know exactly how many iterations you need
while Runs code while a condition is true You don't know how many iterations you need in advance
do-while Runs code at least once, then continues while a condition is true You need the code to execute at least once regardless of the condition

For Loop

The for loop has three parts:

  1. Initialization: Executed once before the loop starts
  2. Condition: Evaluated before each iteration
  3. Update: Executed after each iteration
for (let i = 0; i < 5; i++) {
  console.log(i);
}

While Loop

The while loop continues as long as the specified condition is true. 

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

Do-While Loop

The do-while loop is similar to the while loop, but it always executes the code block at least once before checking the condition. 

let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);

Common Loop Patterns

Looping Through Arrays

let fruits = ["Apple", "Banana", "Orange", "Mango"];

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

Skipping Iterations with continue

for (let i = 1; i <= 10; i++) {
  if (i % 2 === 0) {
    continue; // Skip even numbers
  }
  console.log(i); // Only prints odd numbers
}

Breaking Out of Loops with break

for (let i = 1; i <= 10; i++) {
  if (i === 5) {
    break; // Exit the loop when i is 5
  }
  console.log(i); // Only prints 1, 2, 3, 4
}
