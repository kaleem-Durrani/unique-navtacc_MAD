HTML Form Elements

Note: This page demonstrates various HTML form elements and their attributes. Forms are used to collect user input and send it to a server for processing.

Text Input Fields

Text input fields allow users to enter text data such as names, emails, and passwords.

The required attribute makes this field mandatory.
This field is optional (no required attribute).
The type="email" attribute validates that the input is in email format.
The type="password" attribute masks the input characters for security.

Checkboxes

Checkboxes allow users to select multiple options from a set.

Select your preferred courses:

Users can select multiple checkboxes or none at all.

Radio Buttons

Radio buttons allow users to select exactly one option from a set.

Radio buttons with the same name attribute form a group where only one can be selected.

Number Input

Number inputs allow users to enter numeric values with optional constraints.

The min and max attributes set the allowed range of values.

Range Input

Range inputs allow users to select a value from a range using a slider.

The value attribute sets the default position of the slider.

Select Dropdown

Select dropdowns allow users to choose one option from a dropdown list.

Textarea

Textareas allow users to enter multiple lines of text.

The rows attribute sets the visible height of the textarea.

Submit Button

Submit buttons send the form data to the server for processing.

The type="submit" attribute specifies that this button submits the form.
