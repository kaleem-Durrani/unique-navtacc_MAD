The <p> tag defines a paragraph. The <br> tag inserts a single line break. The <hr> tag creates a horizontal rule (line).

HTML provides specific tags for marking up quotations.

This is a block quote. It is typically indented and used for longer quotations. The <blockquote> tag is a block-level element.
