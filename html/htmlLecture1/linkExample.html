Link Example Page

This page demonstrates how HTML links work. You arrived here by clicking a link on the main page.

Note: In HTML, links are created using the <a> tag (anchor tag). The href attribute specifies the destination of the link.

Types of Links in HTML

  • Internal Links - Link to pages within the same website
  • External Links - Link to pages on other websites
  • Email Links - Open the user's email client (<a href="mailto:example@example.com">)
  • Telephone Links - Call a number on mobile devices (<a href="tel:+1234567890">)
  • Anchor Links - Jump to specific sections within a page (<a href="#section-id">)
