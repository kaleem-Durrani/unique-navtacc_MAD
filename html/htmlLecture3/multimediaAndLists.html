HTML Multimedia and Lists

This page demonstrates various HTML elements for multimedia, lists, and other useful tags.

Text Formatting and Semantic Elements

HTML provides various elements for formatting text and adding semantic meaning.

Span Element

The <span> element is an inline container used to mark up a part of a text: This is a highlighted text within a paragraph.

Cite Element

The <cite> element is used to define the title of a creative work: The Origin of Species by Charles Darwin

Abbreviation Element

The <abbr> element defines an abbreviation or acronym: WWW (hover over to see the full title)

HTML Lists

HTML provides three types of lists: unordered lists, ordered lists, and description lists.

Unordered List

The <ul> element defines an unordered (bulleted) list:

Shopping List:

  • Apples
  • Bananas
  • Milk
  • Bread
  • Eggs

You can change the bullet style with CSS or the type attribute (circle, disc, square).

Ordered List

The <ol> element defines an ordered (numbered) list:

Steps to Make Tea:

  1. Boil water
  2. Add tea leaves
  3. Add sugar
  4. Add milk

You can specify the starting number with the start attribute and the numbering type with the type attribute (1, A, a, I, i).

Ordered List with Custom Start and Type

  1. Fifth item (v)
  2. Sixth item (vi)
  3. Seventh item (vii)
  4. Eighth item (viii)
