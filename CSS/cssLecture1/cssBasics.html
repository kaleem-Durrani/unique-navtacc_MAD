CSS Basics - Cascading Style Sheets

Note: This page demonstrates the three different ways to apply CSS to HTML elements.

Three Ways to Apply CSS

Method Description Usage
1. Inline CSS Applied directly to HTML elements using the style attribute For styling individual elements
2. Internal CSS Defined in the <style> tag in the <head> section For styling a single page
3. External CSS Defined in a separate .css file and linked with <link> tag For styling multiple pages (recommended)

1. Inline CSS Example

Inline CSS is applied directly to HTML elements using the style attribute:

<element style="property: value;">Content</element>

2. Internal CSS Example

Internal CSS is defined in the <style> tag in the <head> section:

<head> <style> selector { property: value; } </style> </head>

The list items above use internal CSS for their font size (20px), defined in the style tag in the head section.

3. External CSS Example

External CSS is defined in a separate .css file and linked with the <link> tag:

<head> <link rel="stylesheet" href="styles.css"> </head>

CSS Specificity and Cascade

When multiple styles target the same element, the browser follows these rules to determine which style to apply:

  1. Specificity: More specific selectors override less specific ones
  2. Source Order: Later styles override earlier styles
  3. Importance: Styles with !important override other styles

Specificity hierarchy (from highest to lowest):

  1. Inline styles
  2. ID selectors (#id)
  3. Class selectors (.class), attribute selectors, and pseudo-classes
  4. Element selectors (p, div, etc.) and pseudo-elements
