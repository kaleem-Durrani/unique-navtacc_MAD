      CSS:

      Cascading Style Sheets.

      There are 3 ways to write CSS.

      1) directly as an attribute, in the opening tag 
      of an html tag.
      
      2) we can write css code in a style tag
      inside the head tag.

      3) we can write CSS in an external .css file and 
      then link it to our html using a link tag inside 
      the head tag.
